# TripAdvisor — The God Build

A hyper-saturated, high-fidelity travel decision platform. Built for travelers who value clarity, cost-transparency, and aesthetic rigor.

## 🚀 Vision
Instead of jumping between 50 tabs of spreadsheets, weather charts, and budget blogs, TripAdvisor consolidates everything into a single, interactive cockpit. It uses smart heuristics (like "Japan-likeness") to help you find destinations that match your vibe without the decision paralysis.

## 💎 Features

### 🌍 Global Intelligence
- **Temporal Awareness Engine**: A real-time global pulse map with day/night terminators, city-specific telemetry, and temporal grid-lines.
- **79+ Countries**: Deep metadata for costs, regional variety, and cultural vibes.
- **Visa Intelligence**: Real-time status based on your citizenship (India, USA, UK, Germany).
- **Currency Cockpit**: Instant conversion across USD, INR, EUR, JPY, and GBP.

### 📊 Smart Discovery
- **Decision Matrix (Explore)**: Filter by budget, vegetarian-friendliness, terrain variety, and "Japan-likeness" (a custom similarity score).
- **Timing Heatmaps**: 12-month seasonality matrices showing the cheapest, best, and peak windows for every destination.
- **Price Trends**: Historical indices (base 100) to visualize when to book.
- **Friction Scoring**: Automated complexity analysis for itineraries.

### 🗺️ Precise Planning
- **Itinerary Engine**: Hand-built, day-by-day routes with kilometer tracking, hourly estimates, and per-day stay costs.
- **Multi-Day Planner**: A private draft tool to build your own custom routes from scratch.
- **Compare Mode**: Side-by-side telemetry for up to 3 countries.
- **Smart Packing**: Context-aware packing lists based on terrain variety (Mountains, City, Beach, etc.).

## 🛠️ Tech Stack
- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS (Custom HSL System)
- **Icons**: Lucide React + Twemoji (High-fidelity cross-platform flags)
- **Charts**: Recharts (Custom telemetry visualizations)
- **Maps**: React-Leaflet (Interactive destination focus)
- **State**: React Context + LocalStorage Persistence

## 🎨 Design Philosophy
- **Notion-Inspired Clarity**: Clean, high-contrast layouts with plenty of whitespace.
- **Google Flights Precision**: Data-dense yet readable telemetry.
- **Cyber-Travel Aesthetic**: Glassmorphism, dynamic scanners, and temporal pulses.
